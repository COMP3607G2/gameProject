# gameProject
COMP3607Project

Prerequisites

    Java 21 or later
    Maven 3.6 or later

1. Download project from the repo and Unzip the project folder
2. Open the project folder in the IDE (preferably VSCode)
3. Confirm Maven and Java installed

Run in the terminal of the IDE

    java -version
    mvn -version

4. Install dependencies and build project

    mvn clean compile

5. Run the Game

Select the run command in the IDE



Playing the game
1. Choose input file format

    Options are CSV, JSON and XML

2. Setup Players

    Enter the number of players, between 1 and 4

    Enter the names of each player

3. Gameplay

    Player selects category

    (Sample games categories are: "Variables & Data Types", "Control Structures", "Functions", "Arrays", and "File Handling")
    

    Player selects the value of the question they wish to answer

    (values are: 100, 200, 300, 400, and 500)


    Player receives question and selects answer

    Player states whether they wish to continue or not

4. Choose report format

    Options are PDF, TXT and DOC
